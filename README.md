# Daily Tracker

## Description
This is a simple web application that allows users to track their daily activities.  
Users can add activities, mark them as completed, and view them in separate sections.  
An admin mode is also available and it allows deletion of activities along with the same functionality as the user.

## Features
- Add activities
- Mark activities as complete
- View active and completed activities
- Admin mode for deleting activities

## Technologies Used
- Python
- Flask
- SQLite
- HTML/CSS
